# Requirements

* Node.js
* Angular-cli

## Node.js
### Download
https://nodejs.org/en/download/
### Versions
```
$ node --version
v6.9.1

$ npm --version
3.10.3
(Or newer version)
```

## Angular-cli
### Download
https://cli.angular.io/
### Version
```
$ ng --version
@angular/cli: 1.0.0-beta.28.3
(Or newer version)
```
